# Feature Reference: Hair

## Pre-Drawing Checklist

- [ ] What's visible? (full hair, tufts under a hat, bangs only?)
- [ ] What color? Uniform or with highlights/lowlights?
- [ ] How does the hair interact with other elements? (flows over forehead, under hat, behind ears?)
- [ ] What style? (individual strands, solid mass, spiky tufts?)
- [ ] Does the hair define the forehead boundary?
- [ ] Are hair outlines separate brush strokes or part of the hair fill?

## SVG Construction

Hair is usually one of the last features added because it interacts with many other elements (hat, ears, forehead).

## Common Mistakes
- Drawing hair that doesn't interact with the hat brim
- Wrong z-ordering (hair should be between face and hat typically)
- Too much detail in individual strands for a cartoon style
