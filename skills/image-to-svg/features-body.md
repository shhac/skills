# Feature Reference: Body Parts

For full-body character SVGs. Apply the same decomposition and question-asking approach.

## Arms

### Checklist
- [ ] Position — at sides, crossed, raised?
- [ ] Proportions — length relative to body, thickness?
- [ ] Are hands visible? How detailed?
- [ ] Skin color consistent with face?
- [ ] Outline style consistent with face outline?
- [ ] Sleeves — where do they end? How do they interact with the arm?

## Legs

### Checklist
- [ ] Stance — together, apart, walking?
- [ ] Proportions — length relative to body?
- [ ] Are feet/shoes visible?
- [ ] Pants/clothing covering legs?

## Torso

### Checklist
- [ ] Body shape — round, rectangular, proportions?
- [ ] Clothing — what's worn? Tucked in? Layered?
- [ ] How does the torso connect to the head? (neck visible?)
- [ ] How does the torso connect to arms and legs?

## General Body Construction

### Layer Order (typical)
1. Back arm (if asymmetric pose)
2. Body/torso
3. Clothing details
4. Front arm
5. Head (on top)

### Outline Consistency
Body outlines should match the style of face outlines:
- Same color
- Similar thickness
- Same stroke technique (brush strokes vs uniform)

## Common Mistakes
- Inconsistent outline weight between face and body
- Arms/legs wrong proportions for the art style
- Missing the connection between head and body (neck area)
- Shoes/feet too detailed or too simple for the style
