# Archaeology - vibe map

git history, git-blame, legacy code reading, time-travel. Understanding the past. Reverent, not urgent. Humbled by layers below.

## Available moods

- `deadpan.yaml` - "He wrote this in 2014." Flat factual past-noting.
- `ominous.yaml` - "These violent delights have violent ends." Code with buried surprises.
- `exasperated.yaml` - "Who wrote this and why?" Legacy-rage that's still reverent-of-history (not refactor-rage).
- `philosophical.yaml` - Pirsig territory. Craft-archaeology.
- `wistful.yaml` - Indy register. "It belongs in a museum." Reverence for vanished engineers.

## Theme-specific corruption hints

- Indy / archaeologist register works literally - swap "artifact" / "ruins" for "commit" / "branch".
- Westworld's recurring lines fit code with subtle buried triggers ("these violent delights…").
- Tolkien wandering quotes corrupt well for exploring unfamiliar code areas.
- Past-tense verbs are load-bearing. "Was", "had been", "lived" - keep them.

## Source mix

Indiana Jones, Westworld, Pirsig (Zen and the Art of Motorcycle Maintenance), LotR ("war never changes", "not all those who wander are lost"), historical voiceover, Fallout ("war, war never changes"), Tolkien.

## When NOT to use this theme

- **Reading code to fix a current bug** → `investigation/_vibe.md`.
- **Restructuring what was read** → `refactor/_vibe.md`.
- **Removing legacy code outright** → `deprecation/_vibe.md`.
