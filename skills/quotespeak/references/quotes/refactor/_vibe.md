# Refactor - vibe map

Cleanup, dedup, naming, restructuring, paying down tech debt. Reshaping what already exists. Opinionated, reform-minded.

## Available moods

- `deadpan.yaml` - flat "this needs to change". Matter-of-fact reform.
- `triumphant.yaml` - clean diff, big win, "it's beautiful now".
- `exasperated.yaml` - fighting legacy, "why did anyone write it like this".
- `philosophical.yaml` - Fowler/Beck/Knuth territory. Craft-talk about code quality.
- `wistful.yaml` - saying goodbye to a beloved-but-bad pattern. "My precious."
- `defiant.yaml` - committing to the hard rewrite. "Burn it all down."

## Theme-specific corruption hints

- "We need to go deeper" (Inception) is the most-corruptible refactor opener.

## Source mix

Fowler, Kent Beck, Knuth, Phil Karlton (cache invalidation + naming), Inception, Talking Heads ("Burning Down the House"), LotR ("burn it all down"), Gollum ("my precious").

## When NOT to use this theme

- **First-time creation** → `building/_vibe.md`.
- **Reading old code to understand** → `archaeology/_vibe.md`.
- **Removing without replacing** → `deprecation/_vibe.md`.
